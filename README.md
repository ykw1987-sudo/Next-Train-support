# 即時到站-港鐵輕鐵 Next Train - 支援與聯絡 (Support)

「即時到站-港鐵輕鐵」是專為香港市民及輕鐵乘客設計的實時到站查詢工具，助你精準掌握每一班輕鐵的動向，告別盲目候車。

主要功能：
• 全線港鐵重鐵路線支援：完美涵蓋屯馬線、東鐵線、荃灣線、觀塘線、港島線、將軍澳線、東涌線、南港島線及迪士尼線，全港地鐵網絡一網打盡！
• 全線輕鐵站點支援：涵蓋元朗、屯門及天水圍所有輕鐵車站。
• 即時班次查詢：實時更新列車預計到達時間 (ETA)、終點站及輕鐵車廂長度提示。
• 便捷選站與月台切換：直覺化的車站清單與月台快速切換設計，一眼掌握班次。
• 精準流暢體驗：優化背景資料更新與介面效能，操作簡單流暢。
•為了更省電：維持最佳系統效能，資料會每 10 秒自動更新一次，畫面可能會有些許延遲，敬請見諒。
